## [1.0.0] - 2025-10-16

### Added
- Initial release of Maxclicks Node.js SDK
- Contact management (create, update, delete, list)
- Object management with custom schemas
- Event tracking with schema validation
- Attribute management for contacts and objects
- Template sending functionality
- API key management
- Full TypeScript support with type definitions
- Comprehensive test coverage
- Builder pattern for contacts and objects
- Error handling with custom error types

### Features
- RESTful API client with fetch-based implementation
- Support for ESM and CommonJS modules
- Automatic environment detection (Node.js 16+)
- Pagination support for list operations
- Custom attribute handling for contacts and objects
- JSON schema validation for events
- Flexible query parameters for filtering and sorting
